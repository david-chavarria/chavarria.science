Definición del problema

El primer paso para hacer historia digital efectiva es definir claramente nuestro problema. Para esto debemos poseer conocimientos previos que permitan comprender las formas en que se pueden utilizar los datos existentes para generar respuestas satisfactorias.

Tratamiento de datos

Podemos dividirlo en dos fases: recolección y modelado. En la recolección aplicamos metodologías informáticas como el web scraping y la digitalización y reconocimiento de información impresa. El modelado contempla una fase previa de normalización y depuración de los datos recolectados, para su posterior procesamiento mediante un algoritmo específico.
