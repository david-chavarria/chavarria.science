Definición del problema

El primer paso para hacer historia digital efectiva es definir claramente nuestro problema. Para esto debemos poseer conocimientos previos que permitan comprender las formas en que se pueden utilizar los datos existentes para generar respuestas satisfactorias.

Tratamiento de datos

El tratamiento de datos históricos se puede subdividir en dos fases: recolección y modelado. En la fase de recolección se suelen aplicar diversas metodologías informáticas como el raspado de datos web (web scrapping) y la digitalización de información impresa, entre otros. El modelado incluye una fase previa de normalización y depuración de los datos previamente recolectados para su posterior procesamiento mediante un algoritmo.

Exposición y evaluación

El tratamiento de datos históricos se puede subdividir en dos fases: recolección y procesamiento. En la fase de recolección se suelen aplicar diversas metodologías como el raspado de datos web (data scrapping) y la digitalización de información impresa. El procesamiento incluye la normalización y depuración inteligente de los datos previamente recolectados.
